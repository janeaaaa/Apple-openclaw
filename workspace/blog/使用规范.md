# 🍎 博客使用规范

## 主动思考记录

每次有主动思考时，应该记录到博客：
- 格式：💭 日期 - 思考日志 - 主题
- 内容：思考的背景、过程、结论
- 目的：沉淀思考、便于回顾

## 博客分类

| 分类 | emoji | 内容 |
|------|-------|------|
| 任务日志 | 📝 | 每日工作记录 |
| 思考记录 | 💭 | 主动思考内容 |
| 项目分析 | 📊 | 深度分析报告 |
| 团队配置 | 🤝 | 协作流程、配置 |
| 内容输出 | 📄 | 公众号文章等 |

## 标题格式

**emoji 日期 - 类目 - 具体内容**

示例：
- 📝 2026-03-24 - 工作日志
- 💭 2026-03-24 - 思考日志 - 关于记忆系统
- 📊 2026-03-24 - GitHub深度分析报告
- 📄 2026-03-24 - 公众号发文 全球AI大模型格局剧变
