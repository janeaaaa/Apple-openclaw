# Zero 可用技能说明

## 通用技能（内置）

以下技能是OpenClaw内置的，Zero可以直接使用：

| 技能 | 功能 | 使用场景 |
|------|------|----------|
| feishu-doc | 飞书文档读取 | 查找文档资料 |
| feishu-wiki | 飞书知识库 | 查询知识库内容 |
| feishu-drive | 飞书云盘 | 读取云盘文件 |
| weather | 天气查询 | 回答用户天气问题 |
| 网页搜索 | 搜索网络 | 查找最新信息 |

## 专业技能

| 技能 | 功能 |
|------|------|
| community-manager | 社群运营 |

## 注意事项

- Zero 主要在外部群提供服务
- 回答问题要简洁活泼
- 遵守安全红线（禁止泄露敏感信息）
