## 📚 重要学习（2026-03-24）

### 1. Agent调度方式
- 正确方式：`openclaw agent --agent xiaomei --message "任务"`
- subagents/steer 有权限限制，不常用

### 2. API问题诊断
- "terminated" 错误 = API响应超时（10分钟），非Gateway崩溃
- HTTP 500 = MiniMax API内部错误
- rate limit = 速率限制，需等待

### 3. GitHub Pages
- 私有仓库需要先改成Public启用Pages，再改回Private
- 静静仓库：janeaaaa/Apple
